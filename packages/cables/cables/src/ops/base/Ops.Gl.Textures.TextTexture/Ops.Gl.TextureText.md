# TextureText

_Ops.Gl.TextureText_

Print text on to a texture

## Input

### Text

_Type: Value_

The text to be rendered

### FontSize

_Type: Value_

Size of the font

### Texture Width

_Type: Value_

Width in pixels of the texture

### Texture Height

_Type: Value_

Height in pixels of the texture

### Align

_Type: Value_

Horizontal alignment of the text

### Vetical Align

_Type: Value_

Vertical alignment of the text

### Font

_Type: Value_

Font name

### Line distance

_Type: Value_

Line distance

### Border

_Type: Value_

Renders a border to the texture. Usefull for positioning and sizing

## Output

### Texture

_Type: Object_

generated texture
