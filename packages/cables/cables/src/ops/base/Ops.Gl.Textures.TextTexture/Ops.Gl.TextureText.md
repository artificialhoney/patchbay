# TextureText

*Ops.Gl.TextureText*

Print text on to a texture

## Input

### Text

*Type: Value*

The text to be rendered

### FontSize

*Type: Value*

Size of the font

### Texture Width

*Type: Value*

Width in pixels of the texture

### Texture Height

*Type: Value*

Height in pixels of the texture

### Align

*Type: Value*

Horizontal alignment of the text

### Vetical Align

*Type: Value*

Vertical alignment of the text

### Font

*Type: Value*

Font name

### Line distance

*Type: Value*

Line distance

### Border

*Type: Value*

Renders a border to the texture. Usefull for positioning and sizing


## Output

### Texture

*Type: Object*

generated texture
